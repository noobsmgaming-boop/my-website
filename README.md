# Kotlin Music Player

A complete Android Studio project for a local music player written in Kotlin and Jetpack Compose with Material Design 3.

## Features

- Reads songs from phone storage with Android 13+ `READ_MEDIA_AUDIO` support and legacy `READ_EXTERNAL_STORAGE` fallback.
- Playlist screen backed by `MediaStore`.
- Search by song title, artist, or album.
- Play, pause, next, and previous controls.
- Background playback through a Media3 `MediaSessionService`.
- Media notification controls generated by Media3 from the active media session.
- Album art loaded from the device media database.
- Dark mode toggle with Material Design 3 colors.
- Android 14 compatible foreground media playback service permissions.
- Google Mobile Ads SDK integration with the official sample AdMob app ID and banner ad unit.

## Open in Android Studio

1. Open this folder in Android Studio.
2. Let Android Studio sync Gradle and install the Android SDK if prompted.
3. Run the `app` configuration on an Android device or emulator.
4. Grant audio and notification permissions when prompted.

## Before publishing

- Replace the sample AdMob app ID in `app/build.gradle.kts` and the sample banner unit in `app/src/main/res/values/strings.xml` with your production AdMob IDs.
- Consider enabling minification and adding signing configuration for release builds.
- Test playback on Android 10 through Android 14+ devices, especially background playback and notification behavior.
